## Clean Architecture + MVVM

<img src ="https://github.com/kudoleh/iOS-Clean-Architecture-MVVM/blob/master/README_FILES/CleanArchitecture+MVVM.png?raw=true" width=600>

```
Application
    - launchScreen
    - AppDelegate
    - SceneDelegate
    - Asset
    
Domain
    - Entities
    - UseCases
    - Interfaces
        - Repositories
        
Presentation
    - Common View
    - Scene # 1
        - View
          - xxViewController
          - xxView
        - ViewModel
          - TestMock UseCase
    - Scene # 2
        - View
          - xxViewController
          - xxView
        - ViewModel

Data
    - Repositories
    - PersistentStorage
```

## SwiftLint 

- custom -> get set 사용 안하기
- 줄간격 2줄 이상 -> 오류
- 강제 언래핑 금지
- camelCase
- class/struct/enum name : 대문자
- Colon Spacing
