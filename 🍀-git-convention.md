## ⚒ commit message

``` terminal=
// (#이슈번호) 타입: 본문
(#2) feat: content
```

| 타입 | 설명 |
| :-- | :-- |
| feat | 새로운 기능 추가 |
| style | 오타 & 들여쓰기 수정 |
| refactor | 기능 변화가 없는 코드수정 |
| fix | 기능 변경 및 오류 해결 |
| docs | 문서 추가 및 수정(`README.md`) |
| test | Test 관련 코드 추가, 수정 |
| chore | 설정 추가 및 수정(숨김파일, 라이브러리) |

</br>
